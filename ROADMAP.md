# Roadmap

## v0.1

- JSON and YAML OpenAPI loader
- MCP TypeScript project generator
- API Key and Bearer auth presets
- generated tests, Docker, and CI
- tag and method filtering
- local `#/components/...` $ref resolution for parameters, request bodies, and schemas
- three example specs and launch-quality README

## v0.2

- response-aware and deprecated-operation selection
- better schema flattening
- better oneOf/anyOf/allOf handling
- HTTP transport template
- generator snapshot coverage

## v0.3

- spec patch files
- plugin hooks
- richer auth presets
- registry publishing helpers

## Product rule

Every roadmap item must improve one of these:

- generation reliability
- output quality
- onboarding speed
- contribution surface
